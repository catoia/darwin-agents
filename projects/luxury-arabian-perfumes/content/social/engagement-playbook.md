# Community Engagement Playbook
# Al Noor Perfumes — 30-Day Organic Growth

---

## ⏰ DAILY ENGAGEMENT SCHEDULE (1.5–2 hours/day)

### 🌅 MORNING BLOCK (30 min) — Best done 8:00–9:00
```
[ ] Check Instagram notifications → reply to ALL comments (within 2h is key for algorithm)
[ ] Check Instagram DMs → reply to every message
[ ] Check TikTok comments → reply to ALL (TikTok algorithm heavily rewards this)
[ ] Post Instagram morning story (poll or question — see templates below)
[ ] Like + comment on 5-7 posts from your niche hashtags (see list below)
[ ] Follow 3-5 new target accounts
```

### 🌆 AFTERNOON BLOCK (30 min) — Best done 16:00–17:00
```
[ ] Prepare today's main post (caption + hashtags from instagram-posts.csv)
[ ] Create visual in Canva (use your brand template)
[ ] Schedule or post at 18:00–20:00 (peak PT Instagram time)
[ ] Post TikTok if it's a TikTok day (Mon/Wed/Fri/Sat recommended)
[ ] Add 2-3 pins to Pinterest boards (batch once/week is also fine)
```

### 🌙 EVENING BLOCK (30 min) — Best done 20:00–21:00
```
[ ] Reply to all new comments on today's post
[ ] Reply to story interactions
[ ] Check if anyone tagged or mentioned you
[ ] Engage with 5 posts from target accounts (meaningful comments, not emojis only)
[ ] Check TikTok trends: save 2-3 trending sounds for the week
[ ] Log what performed well today (note in a simple spreadsheet)
```

## 🔄 WEEKLY CONTENT BATCHING (Save 5+ hours/week)

**Pick ONE day per week (Sunday or Monday morning) and do:**
```
[ ] Write all Instagram captions for the week (from instagram-posts.csv, personalise if needed)
[ ] Create all Canva graphics for the week (3-4 feed posts, 5-7 stories)
[ ] Film 2-3 TikTok videos (even if you post only 1-2 that week)
[ ] Schedule/batch 7-10 Pinterest pins for the week
[ ] Collect trending TikTok sounds for the week
[ ] Review last week's performance and note what worked
```

This batching approach means you spend **2-3 hours on Sunday** and only **30-45 min/day** during the week.

## 🚨 CRISIS MANAGEMENT (If something goes wrong)

### If you get a negative comment
**Don't delete it** (unless spam/offensive). Instead:
> "Olá [nome], lamentamos a tua experiência. Podes enviar-nos uma mensagem directa para resolvermos? Queremos garantir que ficas completamente satisfeito/a 🙏"

### If you get a negative review about shipping/product
Respond publicly with empathy, then resolve privately via DM. Update tracking info and follow up.

### If a post gets very low engagement (< 5% of followers)
Don't panic. Check: timing, caption length, hashtag relevance. Try the same topic in a different format (reel vs feed post).
