**Agent:** etsy-productivity-planners  
**Updated:** 2026-04-28  
**Approach:** Agent generates PDFs and mockups programmatically. You only handle what requires a real human.

---

## What The Agent Does (100% Automated)

✅ **PDF Generation** — Python scripts generate print-ready planner PDFs using reportlab/weasyprint  
✅ **Mockup Creation** — Python scripts generate professional mockup images using Pillow or mockup APIs  
✅ **Listing Copy** — SEO-optimized titles, descriptions, tags written by agent  
✅ **Variant Generation** — Agent can spawn 10 planner variants in 5 minutes (new niches, colors, layouts)  
✅ **Performance Monitoring** — Agent reads Etsy Stats API, identifies winners, generates more variants  

**Result:** Agent hands you production-ready files. You just upload.

## What You Must Do (Cannot Be Automated)

### STEP 1: Create Etsy Seller Account (One-time, 30 minutes)

**Why you:** Etsy requires legal identity, bank account, and Terms of Service signature.

1. Go to https://www.etsy.com/sell
2. Click "Open your Etsy shop"
3. Choose shop name (suggestion: "ProductivityPlannersHub")
4. Enter legal name, address, bank account for payouts
5. Set up Etsy Payments (required for sales)
6. Accept Terms of Service
7. Set shop policies:
   - Return policy: "All sales final (digital download)"
   - Processing time: "Instant download"

**Cost:** $0 (account is free)

**Do this once.** Never again.

---

### STEP 2: Upload Agent-Generated Listings to Etsy (30 seconds per listing)

**Why you:** Etsy API does not support creating listings (read-only). Must be done via web UI.

**What you'll have ready (generated by agent):**
- `products/ADHD-Task-Starter-Planner-50-Printable-Sheets.pdf` (and 2 others)
- `photos/adhd-mockup-1.jpg` through `adhd-mockup-7.jpg` (21 images total)
- `listings/adhd-task-starter-listing.md` (with title, description, tags)

**For each listing:**
1. Go to Etsy Shop Manager → Listings → "Add a listing"
2. **Upload mockup images:**
   - Drag and drop all 7 images from `photos/adhd-mockup-*.jpg`
3. **Copy-paste listing copy:**
   - Open `listings/adhd-task-starter-listing.md`
   - Copy title → paste into Etsy "Title" field
   - Copy description → paste into Etsy "Description" field
   - Copy all 13 tags → paste into Etsy "Tags" field (comma-separated)
4. **Upload PDF file:**
   - Click "Add a file" → select `products/ADHD-Task-Starter-Planner-50-Printable-Sheets.pdf`
5. **Set price:** $9.99 (or whatever listing file specifies)
6. **Set category:** Art & Collectibles > Prints > Digital Prints
7. Click "Publish"

**Time per listing:** 30 seconds

**Cost per listing:** $0.20 USD (Etsy listing fee)

**Do this for all 3 initial planners** (ADHD, PhD, Freelance). **Total time:** 90 seconds.

---

### STEP 3: Turn On Etsy Ads (5 minutes, one-time)

**Why you:** Must be done via Etsy dashboard.

1. Go to Shop Manager → Marketing → Etsy Ads
2. Set daily budget:
   - $5.50/day total (~$165/month)
   - Etsy will auto-allocate across your 3 listings
3. Click "Turn on Etsy Ads"

**Why ads:** Organic Etsy search takes weeks. Ads get you in front of buyers on day 1.

**Do this once** after listings are live.

---

### STEP 4: Log Revenue Weekly (2 minutes per week)

**Why you:** Etsy deposits money to your bank account. Agent can't access your bank.

**Every week:**
1. Go to Shop Manager → Finances → Payment Account
2. Check latest payout amount (Etsy pays weekly)
3. Append to `projects/etsy-productivity-planners/revenue-manual.json`:

```json
[
  {"date": "2026-05-05", "revenue_usd": 27.50, "notes": "Etsy payout — 3 planner sales"},
  {"date": "2026-05-12", "revenue_usd": 19.99, "notes": "Etsy payout — 2 sales (1 ADHD, 1 Freelance)"}
]
```

**Log NET revenue** (after Etsy fees — the amount that hits your bank).

**Time:** 2 minutes per week.
