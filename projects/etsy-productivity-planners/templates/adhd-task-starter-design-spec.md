# ADHD Task Starter Planner — Design Specification

## Format
- **Size:** US Letter (8.5" × 11")
- **Pages:** 50 sheets (printable)
- **Color:** Black & white with optional color accents (teal/coral)
- **Font:** Clean sans-serif (Poppins or Inter), 11-14pt

## Page Templates (50 total)

### Daily Task Starter Page (30 pages)
**Layout:**
```
┌─────────────────────────────────────────────┐
│ Date: ______  Energy Level: ○ Low ○ Med ○ Hi│
├─────────────────────────────────────────────┤
│ TODAY'S PRIORITY TASK                        │
│ ┌─────────────────────────────────────────┐ │
│ │ Task: ____________________               │ │
│ │                                          │ │
│ │ Why it matters: ____________             │ │
│ │                                          │ │
│ │ First tiny step (5 min): ___________    │ │
│ │                                          │ │
│ │ Next 3 micro-steps:                     │ │
│ │ ○ ______________________________        │ │
│ │ ○ ______________________________        │ │
│ │ ○ ______________________________        │ │
│ └─────────────────────────────────────────┘ │
├─────────────────────────────────────────────┤
│ HYPERFOCUS SESSION LOG                       │
│ Started: __:__ | Ended: __:__ | Duration: __│
│ What I accomplished: _______________________│
│ ___________________________________________  │
│                                              │

├─────────────────────────────────────────────┤
│ BRAIN DUMP (worries, distractions, ideas)   │

│ ___________________________________________  │

│                                              │
│ WIN OF THE DAY (even if small):             │

└─────────────────────────────────────────────┘
```

### Weekly Overview (10 pages)
- 7-day glance with energy tracking
- Top 3 non-negotiables for the week
- Procrastination triggers to watch for
- Weekly reflection: what worked, what didn't

### Task Decomposition Worksheets (10 pages)
- Break down 1 overwhelming task into 20+ micro-steps
- Identify blockers and workarounds
- Accountability buddy reminder section

## Key Features
- **No guilt language** — focus on progress, not perfection
- **Energy-aware** — tasks sorted by required focus level
- **Visual cues** — checkboxes, circles, and simple icons
- **Dopamine tracking** — celebrate small wins explicitly
