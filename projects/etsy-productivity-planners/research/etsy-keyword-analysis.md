## Methodology

Since live API access to eRank/Marmalead is unavailable, this analysis uses:
1. **Etsy autocomplete patterns** — well-documented keyword suggestions Etsy surfaces based on real buyer queries
2. **Google Trends directional data** — relative popularity comparisons between keyword variants
3. **Competitor title/tag mining** — reverse-engineered from top-ranking Etsy listings in each category
4. **Long-tail intent analysis** — identifying buyer-intent phrases with lower competition but high conversion

**Opportunity Score** = Volume (1–5) × (6 − Competition level 1–5) — prioritizes high-volume, low-competition gaps
